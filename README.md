# Animal Classifier - Dog vs Cat

A modern web application that uses a Teachable Machine model to distinguish between dogs and cats in real-time.

## Features

- **Real-time Classification:** Uses your webcam to identify animals.
- **Image Upload:** Upload photos of your pets to see the results.
- **Interactive UI:** Dynamic results display with confidence bars.
- **Dark/Light Mode:** Seamless theme switching.
- **Mobile Responsive:** Works perfectly on all screen sizes.

## Built With

- **HTML5/CSS3** (Modern Baseline features)
- **JavaScript** (ES6+)
- **TensorFlow.js**
- **Teachable Machine Image Library**

## Model

The application uses a custom Teachable Machine model trained to recognize dogs and cats.
Model URL: [https://teachablemachine.withgoogle.com/models/57_063iBZ/](https://teachablemachine.withgoogle.com/models/57_063iBZ/)
