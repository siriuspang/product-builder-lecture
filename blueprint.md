
# Lotto Number Generator

## Overview

This project is a simple web application that generates and displays a set of random lottery numbers. The application is built using modern web development practices, including the use of Web Components to create a reusable and encapsulated UI.

##- Features

- **Lottery Number Generation:** Generate a set of 6 unique random numbers between 1 and 45.
- **Modern UI:** A clean and visually appealing interface for displaying the generated numbers.
- **Web Components:** The application uses a custom element (`lotto-numbers`) to encapsulate the display of the lottery numbers, making the code more modular and reusable.
- **Responsive Design:** The layout adapts to different screen sizes, ensuring a good user experience on both desktop and mobile devices.

## Current Plan

- **HTML Structure:** Create the basic HTML structure for the application, including a title, a container for the lottery numbers, and a button to trigger the number generation.
- **CSS Styling:** Apply modern CSS to style the application, including a color palette, typography, and layout.
- **JavaScript Logic:** Implement the lottery number generation logic and create a custom element to display the numbers.
